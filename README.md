This project is the backend API service for [lolisland.us](https://github.com/101island/lolisland.us). Built with [Cloudflare Workers](https://workers.cloudflare.com/) and [Hono](https://hono.dev/) framework.

## Prerequisites

Please use [Bun](https://bun.sh/) as the package manager.

## Installation

```bash
bun install
```

## Local Development

Start local development server:

```bash
bun run dev
```

## Code Style

Use [Biome](https://biomejs.dev/) for linting and formatting.

- **Check Code**: `bun run check`
- **Format Code**: `bun run format`

Please make sure to check code format before committing.
