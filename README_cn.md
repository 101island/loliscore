本项目是 [lolisland.us](https://github.com/101island/lolisland.us) 的后端 API 服务。基于 [Cloudflare Workers](https://workers.cloudflare.com/) 和 [Hono](https://hono.dev/) 框架开发。

## 环境准备

请使用 [Bun](https://bun.sh/) 作为包管理器。

## 安装依赖

```bash
bun install
```

## 本地开发调试

启动本地开发服务器：

```bash
bun run dev
```

## 代码规范

使用 [Biome](https://biomejs.dev/) 进行代码检查和格式化。

- **检查代码**: `bun run check`
- **格式化代码**: `bun run format`

提交前请务必检查代码格式。
